# Hello Docker 🚀

A beginner DevOps project demonstrating how to containerize a simple Flask application, build a Docker image, and push it to Docker Hub.

---

## 📦 Project Overview
This project shows:
- How to write a `Dockerfile` for a Python Flask app.
- How to build and run a Docker image locally.
- How to tag and push the image to Docker Hub (`jaysam2026/hello-docker:v1`).
- How to set up GitHub for version control.

---

## 🛠 Technologies Used
- **[Python Flask](ca://s?q=Python_Flask_basics)** – lightweight web framework.
- **[Docker](ca://s?q=Docker_basics_project)** – containerization.
- **[Git & GitHub](ca://s?q=Initialize_Git_repository_locally)** – version control and collaboration.
